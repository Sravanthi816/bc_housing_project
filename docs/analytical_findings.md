# Analytical Findings

## Overview

This document contains key analytical observations discovered during the Vancouver Housing Analytics project.

---

# Initial Findings

## Housing Affordability

Mortgage burden percentages increased significantly in high-demand regions.

Regions with higher housing prices generally showed:

- higher mortgage burden
- lower affordability
- stronger long-term growth trends

---

# Housing Type Analysis

Detached homes consistently showed:

- highest median housing prices
- strongest affordability pressure

Condos generally showed:

- lower housing prices
- relatively lower mortgage burden

---

# Regional Trends

Some CMA regions experienced:

- rapid housing price appreciation
- higher growth percentages
- stronger long-term upward trends

---

# Time-Series Trends

Window-function analysis revealed:

- cumulative growth trends
- moving average smoothing patterns
- long-term market growth behavior

---

# SQL Analytics Observations

Analytical SQL techniques provided insights into:

- growth behavior
- affordability changes

- cumulative market trends
