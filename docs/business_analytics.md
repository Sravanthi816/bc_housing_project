## Important SQL Concepts

### Window Functions Preserve Rows
Window functions calculate analytical values without collapsing rows.

### GROUP BY Defines Analytical Grain
GROUP BY determines the aggregation level of analysis.

### Correlated Subqueries
Subqueries that depend on outer query rows.

### EXISTS
Checks whether matching rows exist.

### ANY vs ALL

ANY:
- comparison with at least one value

ALL:
- comparison with every value
