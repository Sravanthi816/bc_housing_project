# Dataset Description

## Overview

This dataset contains housing affordability information for multiple CMA (Census Metropolitan Area) regions across Canada.

The dataset is used to analyze:

- housing affordability
- regional housing trends
- mortgage burden
- housing price growth
- affordability comparisons
- time-series market analysis

Source:
https://catalogue.data.gov.bc.ca/dataset/b-c-housing-affordability

# Main Table

housing_affordability

# Important Columns

| Column Name | Description |
|---|---|
| cma_id | Census Metropolitan Area ID |
| cma_name | Name of CMA region |
| year | Year of observation |
| quarter | Quarter of observation |
| housing_type | Type of housing |
| median_housing_price | Median housing price |
| median_after_tax_annual_family_income | Median family income after tax |
| prime_interest_rate | Prime interest rate |
| mortgage_payment_percent_income | Percentage of income spent on mortgage |
| historic_payment_percent_income | Historical payment burden |
| number_of_resales | Number of housing resales |
